M4ED - mobile learning platform

m4ed is mobile optimized web based system that enables the creation of m-learning services that work on most of the available Mobile Internet connected devices. M4ed is an outcome of the Human Learning Project that has been conducted in co-operation with Aalto University's Department of Media Technology 2012-2013

demo.m4ed.com Learn more of m4ed

Architecture and Developer Info

m4ed is open sourced and released under the MIT license. The source code is hosted in github and the technical documentation is maintained in the github project wiki.

Demo Site

We have a demo site running at demo.m4ed.com where you can access the latest master branch build. The databases are wiped empty without warning, so we take no responsibility for any loss of your edits there. Registration is needed, but no real credentials are required and no cleartext passwords are stored. Remember to try out the editor!
